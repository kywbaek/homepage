Venues Nearby

 http://venues-nearby.youngwookbaek.com

A single page web application featuring a map of the area user provided with responsive display. The app provides a list of recommended venues within a variety of categories near the area. Users can filter venues by their categories and get the information about each venue from infowindows of its markers on the map. Built with JavaScript library Knockout, Google Maps APIs, and Foursquare API.

Catalog App

 http://catalog.youngwookbaek.com

I used the Python framework Flask to develop a RESTful web application that provides a list of items within a variety of categories. I also successfully implemented Google and Facebook OAuth authentication system and used the Python SQL toolkit SQLAlchemy to build the underlying database.

MLB Team Stats' Relative Importance

 https://github.com/kywbaek/mlb_stat_importance

I am a huge baseball fan. I usually check the scores and stats for my favorite teams after each game. Watching the games is very much enjoyable itself but I often imagine myself being a general menager building and leading mlb team. In this project, I built linear regression models predicting team's winning percentage from team stats and compare the "importance" of each stat by its coefficient. I used statistics data from seanlahman.com.

Bible Verses Classifier

 https://github.com/kywbaek/bible_verses_classifier

Bible has two main parts. Old Testament and New Testament. They differ in many ways: background, authors, theme, etc. Each Testament consists of books which consist of more than 30000 verses in total. In this project, I built a model classifying a given bible verse to the right Testament using Naive Bayes.
